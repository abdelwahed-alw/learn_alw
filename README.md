Salearn – AI Language Tutor 🤖🌍
Salearn is a modern, high-performance mobile application built with Flutter. It acts as a personal AI-powered language tutor designed to help users master new languages through interactive scenarios, real-time feedback, and diverse study topics.
✨ Features
 * AI-Driven Feedback: Get instant, detailed corrections and natural-sounding suggestions for your sentences (as seen in the "AI Feedback" section).
 * Diverse Study Topics: Practice across multiple categories:
   * Daily Conversations, Philosophy, Movies & TV Shows.
   * Business & Finance, Science & Nature, Technology, and more.
 * Bilingual Support: Seamlessly switch between languages (e.g., Arabic to English) for a tailored learning experience.
 * Modern UI/UX: A sleek, productivity-inspired dark theme with intuitive navigation and responsive components.
 * Smart Suggestions: "Better ways to say it" feature providing multiple natural alternatives to user inputs.
 * API Integration: Securely connect your own AI keys to power the tutoring engine.
🛠 Tech Stack
 * Framework: Flutter (Multi-platform UI toolkit)
 * Language: Dart
 * State Management: (e.g., Provider / Riverpod / GetX)
 * AI Backend: Integration with LLM APIs (OpenAI/Gemini) for linguistic analysis.
 * Architecture: Clean Architecture for scalability and maintenance.
🚀 Getting Started
Prerequisites
 * Flutter SDK installed.
 * A mobile emulator or a physical device.
 * An API key for the AI backend.
Installation
 * Clone the repository:
   git clone https://github.com/yourusername/salearn.git
cd salearn

 * Install dependencies:
   flutter pub get

 * Run the app:
   flutter run

📁 Project Structure Highlights
 * lib/models: Data structures for Topics and Feedback.
 * lib/screens: UI screens (Dashboard, Chat, Settings).
 * lib/services: API integration and AI logic.
 * lib/widgets: Reusable UI components like Topic Cards and Feedback Bubbles.
